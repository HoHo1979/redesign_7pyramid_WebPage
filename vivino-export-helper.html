❓ How to Use

  1. Click "Load Data from Command Center" to retrieve your collected URLs
  2. See how many wines you've completed
  3. Click "Export as CSV" to get a properly formatted CSV file
  4. Copy the output (or download via file system)
  5. Use this CSV with the merge script: node merge-vivino-images.js

⚙️ Technical Details

Where data comes from: Browser localStorage from the Command Center

What gets exported: Wine names, vintages, regions, statuses, and image URLs you've collected
