🍷 Vivino Command Center

Automated wine search helper - opens Vivino searches one by one

⚡ Quick Setup (First Time Only)

  1. Install Tampermonkey browser extension (Chrome/Firefox/Edge/Safari)
  2. Download & install the vivino-helper.user.js script (adds copy buttons on Vivino)
  3. Come back here and load your wine list using "Load Wine List from CSV"
  4. Click "Open Next Wine Search" to start - it opens Vivino with the wine name pre-filled
  5. Find the wine, click the bottle image → use the "Copy Image URL" button that appears
  6. Return here and paste the URL, then click "Next Wine"
  7. Repeat until done!
