## Understanding the Format

### Field Descriptions

**Wine Name:** English name of the wine, often including producer and vintage

**Chinese Name:** Traditional Chinese name with packaging information (箱 = box, 瓶 = bottle)

**Product ID:** Unique identifier in the Seven Pyramid inventory system

**Category:** Wine region or classification
- Example: Saint-Emilion, Pomerol, 一級酒莊 (First Growth)

**Score:** Critic ratings from various reviewers
- RP = Robert Parker/Advocate
- JS = James Suckling
- WE = Wine Enthusiast
- WS = Wine Spectator
- JA = Jancis Robinson
- DE = Decanter

**Stock Quantity:** Current inventory
- 瓶 = bottle
- 套 = set/case

**Cost Price:** Price in Taiwan New Dollars (NT$)

**Winery/Producer:** Link to official winery website

**Region:** Wine producing region (e.g., Saint-Emilion, Pomerol)

**Winery Image:** Direct link to winery image (primarily from Vivino database)

**Vivino Link:** Direct search link to Vivino database for the specific wine

## Image URL Format

### Vivino Images
Format: `https://images.vivino.com/thumbs/[UNIQUE_ID]_375x500.jpg`

Example: `https://images.vivino.com/thumbs/Z0CY27K1T12To3RXLLwhGg_375x500.jpg`

**Characteristics:**
- High quality product photography
- Standardized size (375x500 pixels)
- Reliable hosting (Vivino servers)
- Suitable for commercial use with attribution

### Official Winery Websites
Format: `https://www.[producer-name].com` or similar

Examples:
- https://www.chateau-ausone.com
- https://www.chateau-margaux.com
- https://www.lafite.com

---

## Search Links

For wines without direct image URLs, two searchable links are provided:

### Wine-Searcher
Format: `https://www.wine-searcher.com/find/[WINE_NAME]`

Example: `https://www.wine-searcher.com/find/CH%20AUSONE%202020`

**Benefits:**
- Aggregates prices across international retailers
- Shows bottle images and tasting notes
- Provides purchase options
- Highly reliable for rare wines

### Vivino
Format: `https://www.vivino.com/search?q=[WINE_NAME]`

Example: `https://www.vivino.com/search?q=CH%20AUSONE%202020`

**Benefits:**
- Community wine database
- User reviews and ratings
- Bottle images
- Food pairing suggestions
- Social wine community

## Using These Links

### For Website Display
Copy the image URLs directly to your HTML:
```html
<img src="https://images.vivino.com/thumbs/Z0CY27K1T12To3RXLLwhGg_375x500.jpg" 
     alt="Château Ausone">
```

### For Database Integration
Parse the markdown file to extract:
- Wine metadata (name, Chinese name, product ID)
- Pricing information
- Critic scores
- Producer information
- Image URLs

### For Cross-Referencing
Use the Vivino/Wine-Searcher links to verify:
- Correct vintage
- Producer accuracy
- Current market prices
- Availability information
