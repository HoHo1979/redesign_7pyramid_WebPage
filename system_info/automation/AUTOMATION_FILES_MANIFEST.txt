🎯 HOW THE SYSTEM WORKS
================================================================================

STEP 1: SETUP (One-time)
   └─ Install browser extension & helper script
   └─ Open Command Center
   └─ Load CSV file

STEP 2: COLLECTION (Main workflow, per wine)
   1. Click "Open Vivino Search"
      → New tab opens with wine name pre-filled
   2. Find the wine in search results
      → Click correct vintage
   3. Copy image URL
      → Hover image → Click copy button
      → OR right-click → Copy Image Link
   4. Return to Command Center
      → Paste URL (Ctrl+V)
   5. Save & Next
      → Press Enter OR click button
      → Auto-moves to next wine
   6. REPEAT for all 164 wines!

STEP 3: EXPORT (Optional but recommended)
   └─ Open vivino-export-helper.html
   └─ Load data from Command Center
   └─ Export as CSV
   └─ Download or copy file

STEP 4: MERGE (Final automation)
   └─ Run: node merge-vivino-images.js
   └─ Script automatically:
      ├─ Reads your CSV
      ├─ Matches wines by name+vintage
      ├─ Updates wines-data-template.json
      ├─ Creates backup
      └─ Shows summary report

DONE! 🎉
   ├─ All 164 wines have image URLs
   ├─ Wine database automatically updated
   ├─ Ready to deploy
   └─ 2-3 hours faster than manual!
