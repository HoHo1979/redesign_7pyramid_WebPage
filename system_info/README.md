# system_info

This folder contains restored project reference documents recovered from the last git commit and reorganized for easier browsing.

## Layout

- `automation/` - automation notes, manifests, quick references
- `deployment/` - deployment checklists, guides, deliverable notes
- `design/` - design and redesign planning material
- `seo/` - SEO, GEO, search console, and discovery notes
- `images/` - wine image workflow and reference notes
- `vivino/` - Vivino-related workflow and tracking files
- `business/` - sales, catalog, and business reference material
- `assistant/` - assistant/tooling guidance documents
- `deploy_reference/` - non-runtime deploy reference docs restored from `deploy/`
- `archive/` - older summaries kept for reference

## Notes

- `.specify/` was intentionally left unchanged.
- Runtime site files remain under `deploy/`.
- Standard project files like `LICENSE.txt`, `package.json`, and `package-lock.json` remain at the repository root.
